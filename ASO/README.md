# EXPLICACIÓ
Aquesta assignatura explica els conceptes fonamentals per poder administrar Linux. Tasca important si vols ser ``sysadmin``. <br>
Administrar significa: Gestió de Discs, Gestió usuaris, Configuració de serveis, Creació de Scripts, ... <br>
Tot i que és digui en el títol "Sistemes Operatius" només es toquen conceptes d'UNIX. <br>

## Teoria
S'expliquen els conceptes/serveis necessaris i més comuns. <br>
Hi ha molta informació per internet.
